# Blog de Manutenção de Computadores

Um blog educativo sobre manutenção e reparo de computadores, cobrindo limpeza, diagnóstico e upgrades simples.

## 📄 Páginas

- **index.html** - Página inicial com introdução ao blog
- **posts.html** - Posts sobre limpeza, diagnóstico e upgrades
- **contato.html** - Formulário de contato com links diretos

## 🎨 Tecnologias

- CSS3
- JavaScript
- Python (Flask para backend opcional)
